# yamlaug
YAML Augmentation tool (non-destructive YAML upgrader)

This is a placeholder repository for the yamlaug project, which aims to provide a tool for upgrading and merging YAML files. The project is currently in the planning stage, and the CLI and core functionality will be implemented soon.

- [yamlaug on Github](https://github.com/taishi-hashimoto/yamlaug)
- [yamlaug on PyPI](https://pypi.org/project/yamlaug/)

## Install

```bash
pip install yamlaug
```

### Dependencies

- [ruamel.yaml](https://pypi.org/project/ruamel.yaml/)

## Usage

```bash
yamlaug --help
```
